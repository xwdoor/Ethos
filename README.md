# Ethos

高逼格朋友圈「图片/格言」编辑器

## 开始

最近会有事没事会在朋友圈发动态：写一句格言警句，配上一张意境美图。因为懒，我总是在想，图片能够自动随机网上选择该多好，能不能直接将文字写在图片上，能不能一键分享到朋友圈，再配上二维码，那感觉，妙。。。

于是想要开发一个 App，顺便写一个系列文章，记录 App 的开发过程。

GIF 录制工具：「GIF Brewery 3」
